# 🛒 E-commerce Furniture Data Analysis

## 🔹 Objective
Analyze furniture sales data to understand pricing trends, discounts, and factors affecting product sales.

## 🔹 Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (Machine Learning)

## 🔹 Project Workflow
1. Data Collection
2. Data Cleaning (handling missing values, formatting)
3. Exploratory Data Analysis (EDA)
4. Feature Engineering (discount percentage)
5. Model Building:
   - Linear Regression
   - Random Forest Regressor
6. Model Evaluation (MSE, R² Score)

## 🔹 Key Insights
- Discounts significantly influence sales
- Pricing strategy affects product demand
- Random Forest performed better than Linear Regression
